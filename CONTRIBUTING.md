## How Can I Contribute?

### Reporting Bugs
- Ensure the bug was not already reported by searching on GitHub under [Issues](https://github.com/abhayjsx/NirvanaSrc/issues).
- If you're unable to find an open issue addressing the problem, [open a new one](https://github.com/abhayjsx/NirvanaSrc/issues/new).
- Be sure to include a title and clear description, with as much relevant information as possible.

### Development Setup

1. Fork the repository and clone it locally.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Copy `.env.example` to `.env` and fill in the required values.
4. Start the development server:
   ```bash
   npm run dev
   ```

### Pull Request Process

1. Fork the repository and create your branch from `main`.
2. Make your changes.
3. Ensure your code follows the project's code style.
4. Update the documentation as needed.
5. Test your changes thoroughly.
6. Submit a pull request with a clear description of your changes.

## Code Style

- Use 2 spaces for indentation.
- Use single quotes for strings.
- Use camelCase for variable and function names.
- Use PascalCase for class names.
- Use descriptive variable and function names.
- Add comments to explain complex logic.
- Keep lines under 100 characters.
